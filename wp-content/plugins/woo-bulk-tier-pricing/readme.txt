Define quantity-based discounts per product or category. Display tiered prices on product pages and apply discounts automatically in the cart/checkout.

== Description ==
- Admin UI for rules: product or category scope, enable/disable, tiers with percentage/fixed amounts, priority.
- Product page: tier table updates when quantity changes.
- Cart & checkout: adjusts per-unit prices; order items reflect discounted totals.

== Installation ==
1. Upload `woo-bulk-tier-pricing` to `/wp-content/plugins/`
2. Activate via Plugins
3. Go to WooCommerce → Bulk & Tier Pricing → Add Rule
4. Create tiers and publish
