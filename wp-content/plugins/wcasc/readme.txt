Checkout Add-Ons + Slide-In Sidebar Cart for WooCommerce with full admin controls.

== Description ==
- Checkout add-ons section with admin product/category selection and preview.
- Slide-in sidebar cart: cart icon, items with qty controls, remove, free shipping progress, and "You'll love these" slider.
- Customizable colors, widths, fonts, prices display, subtotal, and more.

== Installation ==
1. Upload `wcasc` to `/wp-content/plugins/`.
2. Activate plugin in WordPress > Plugins.
3. Configure under WooCommerce > WC Add-Ons & Sidebar Cart.
4. Create add-on groups under WooCommerce > Checkout Add-Ons.
